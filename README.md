# Лабиринт

Простая игра-лабиринт, разработанная на Flutter.

## Требования

Перед запуском убедитесь, что установлены:

- Flutter SDK 3.41.2 или новее
- Dart SDK
- Android Studio или Visual Studio Code
- Android SDK

Проверить окружение можно командой:

```bash
flutter doctor
```

---

## Установка проекта

Клонируйте репозиторий:

```bash
git clone https://github.com/tagir404/maze_game.git
```

Перейдите в папку проекта:

```bash
cd maze_game
```

Установите зависимости:

```bash
flutter pub get
```

## Запуск приложения

Посмотреть доступные устройства:

```bash
flutter devices
```

Запустить приложение:

```bash
flutter run
```

Или указать конкретное устройство:

```bash
flutter run -d <device_id>
```

### APK

```bash
flutter build apk
```

### Android App Bundle

```bash
flutter build appbundle
```
